ESTE PROYECTO TIENE EL PATRON DE MVC
SE CREÓ UN MINI FRAMEWORK
GENERAR LAS DEPENDENCIAS DE NPM 'npm i'
GENERAR EL COMPOSER 'composer init'
GENERAR EL AUTOLOAD Y ACTUALIZAR COMPOSER
CONTIENE GULP 'gulp'
INICIA DESDE EL index.php DE LA CARPETA public
SE GENERÓ UN PUERTO PARA ABRIR EL PROYECTO DESDE LA CARPETA public (php -S localhost:3000)

LA BASE DE DATOS SE LLAMA app_salon, CONTIENE LAS TABLAS usuarios, servicios, citas (FK -> usuarios), citasServicios (FK -> citas, FK -> servicios)
LAS FK DE LAS TABLAS citas, citasServicios TIENEN RESTRICCIÓN DE INTEGRIDAD REFERENCIAL EN set null


TRABAJA CON UN ROUTING, AUTOLOAD GENERADO POR COMPOSER PARA LOS NAMESPACE, ACTIVE RECORD PARA TRABAJAR CON LA BD
USAMOS LA LIBRERIA phpmailer PARA ENVIO DE EMAILS, LO TRABAJAMOS DESDE UNA CLASE Email, PARA CONSTRUIR LOS EMAILS A ENVIAR
